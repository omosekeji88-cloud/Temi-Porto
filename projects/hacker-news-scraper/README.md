# Hacker News Scraper

A Python web scraping project that collects stories from Hacker News, extracts titles, links and vote counts, then filters for high-scoring posts.

## Key skills

- Python scripting
- Web scraping
- BeautifulSoup HTML parsing
- Data extraction and filtering
- Ranked output generation

## Why it matters for data analysis

The project turns raw web page HTML into structured records, then filters those records into a more useful dataset. This is a common data analyst workflow: collect, clean, transform and rank information.

## Run

```bash
python hacker_news_scraper.py
```
